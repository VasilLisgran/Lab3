## Постановка задания

Разработать программу на языке Java для статистического анализа числовых рядов. Программа должна предоставлять функционал для работы с массивом целых чисел, включая добавление элементов, сортировку и вычисление основных статистических характеристик.

## Описание функционала 

1. Создание пустого списка чисел, предназначенного для анализа
2. Вывод отсортированного списка (по возрастанию)
3. Вставка нового числа в список с автоматическим сохранением порядка (по воз-
растанию)
4. Вывод отсортированного списка (по убыванию)
5. Вывод частотного ряда. Отображение списка чисел в формате: «число - количество раз, которое оно встре-
чается».
6. Размах числового ряда
7. Среднее арифметическое
8. Среднее геометрическое
9. Среднее гармоническое
10. Нахождение моды
11. Нахождение медианы
12. Вычисление дисперсии
13. Среднеквадратичное отклонение
14. Дисперсия альтернативного признака
15. Вывод всех характеристик ряда
